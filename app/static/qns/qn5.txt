                                                Not All Primes Wear Capes

Given a natural number(N), determine whether the number is prime or not. Output "1" if the given natural number is prime
else Output "0".


CONSTRAINT:
    1 ≤ N ≤ 10^3

Example : 

	INPUT    :   5
	OUTPUT   :   1

	INPUT    :   10
	OUTPUT   :   0
