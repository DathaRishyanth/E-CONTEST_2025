                                                            EMORDNILAP?

You are given a string S consisting of lowercase or uppercase English letters. 
Your task is to determine if the string is palindrome or not. 
A palindrome is a string that reads the same backward or forward.

INPUT:
A single string S, where S contains only letters and numbers.

OUTPUT:
Print YES if the string is a palindrome, otherwise print NO.

CONSTRAINT:
    1 ≤ |S|(Length of the string) ≤ 10^3

EXAMPLE:
    INPUT   :   rotator
    OUTPUT  :   YES

    INPUT   :   ABBa
    OUTPUT  :   NO

    INPUT   :   toshit
    OUTPUT  :   NO
