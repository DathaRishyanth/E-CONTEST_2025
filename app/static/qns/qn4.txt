                                                            EMORDNILAP?

You are given a string S consisting of lowercase or uppercase English letters. Your task is to determine if the string is palindrome or not. 
A palindrome is a string that reads the same backward or forward.

INPUT:
A single string S, where S contains only letters and numbers and |S| is the length of the string. The string contains no whitespace characters.

OUTPUT:
Print YES if the string is a palindrome, otherwise print NO.

CONSTRAINT:
    1 ≤ |S| ≤ 1000

EXAMPLE:
    INPUT   :   rotator
    OUTPUT  :   YES

    INPUT   :   ABBa
    OUTPUT  :   NO

    INPUT   :   toshit
    OUTPUT  :   NO
